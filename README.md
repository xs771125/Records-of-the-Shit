# 屎记

> 太史公曰：每日一记，可知肠胃之兴衰。

一款古风便便追踪 Android 应用，以《史记》笔法记录每日肠道健康。

## 功能

- **历书** — 日历视图，记录密度以竹简色深浅标识，已记录日期标记 💩
- **本纪** — 时间线列表，滑动删除（可撤销），每笔记录自动生成"太史公曰"文言评语
- **统计** — 记录总数、连续天数、黄金时段、Bristol 分布、颜色分布、记录跨度
- **成就** — 15 项功勋称号（史官 → 太史令），解锁条件涵盖连击、总量、形态多样性
- **导出** — 生成竹简风格的 PDF「史册」，支持分享

## 记录项

| 字段 | 说明 |
|------|------|
| 日期时间 | 可手动设置，支持补记 |
| Bristol 类型 | 1-7 级，从"分离硬块"到"水状" |
| 颜色 | 棕色 / 深褐色 / 黑色 / 绿色 / 黄色 / 灰白色 / 血色 |
| 气味 | 1-5 星 |
| 顺畅度 | 1-5 星 |
| 备注 | 自由填写 |

## 技术栈

- Kotlin + Jetpack Compose
- Room (SQLite)
- Material 3 (Material You)
- Compose Navigation
- Android PdfDocument

## 构建

```bash
./gradlew assembleDebug
```

要求：JDK 21+、Android SDK API 26+
